Renderizado condicional a traves del ternario

Vamos hacer un ejemplo de como renderizar una parte de HTML segun una condicion a traves de un condcional ternario que manejamos en JavaScript

El ejemplo trata de una viariable que maneja la edad y si es mayor de edad o menor de edad


            import React from 'react'
            import { Fragment } from 'react'
            
            export default function App() {
            
              let edad = 18
            
              return (
                <>
                  <div>
                    <h1>Renderizado Condicional a traves de funcion</h1>
                    {edad >= 18 ? <h2>Eres mayor de edad</h2> : <h2>Eres menor de edad</h2>}
                  </div>
                </>
              )
            }

Renderizado condicional a traves de una funcion

Vamos hacer un ejemplo de como renderizar una parte de HTML segun una condicion a traves de un condicional clasico que retorna un valor HTML dentro de una funcion que manejamos en JavaScript

El ejemplo trata de una viariable que maneja la edad y si es mayor de edad o menor de edad


                import React from 'react'
                import { Fragment } from 'react'
                export default function App() {
                
                  let edad = 18
                  let content=""
                
                  // Funcion que devuelve un contenido html de acuerdo al estado que evalua el condicional
                  function contenido(){
                    if (edad>=18){
                       return content = <h2>Eres mayor de edad</h2>
                    }else{
                      return content = <h2>Eres menor de edad</h2>
                    }
                  }
                  
                  return (
                    <>
                      <div>
                        <h1>Renderizado Condicional clasico a traves de una funcion</h1>
                        {contenido()}
                      </div>
                    </>
                  )
                }

Renderizado con Operador Lógico &&

Vamos hacer un ejemplo de como renderizar una parte de HTML segun una condicion a traves de un operador logico && se usa solo cuando queremos mostrar la parte verdadera


              import React from 'react'
              import { Fragment } from 'react'
              
              export default function App() {
              
                let edad = 18
                let content=""
              
                return (
                  <>
                    <div>
                      <h1>Renderizado Condicional con operador logico &&</h1>
                      {edad >= 18 && <h2>Eres mayor de edad</h2>}
                    </div>
                  </>
                )
              }

En React, cuando necesitas renderizar un condicional con varias líneas de HTML, puedes usar:

  1. Un operador ternario dentro de () para retornar múltiples líneas.
  2. Una estructura if antes del return para definir qué renderizar.
  3. Un componente separado Un componente separado para manejar la lógica condicional.

Ejemplo 1: Usando operador ternario con ()

Si el usuario está autenticado, se muestra su perfil; de lo contrario, se muestra un mensaje de inicio de sesión.


            function App() {
              const isLoggedIn = false;
            
              return (
                <div>
                  {isLoggedIn ? (
                    <div>
                      <h1>Bienvenido, Usuario</h1>
                      <p>Nos alegra verte de nuevo.</p>
                    </div>
                  ) : (
                    <div>
                      <h1>Acceso Denegado</h1>
                      <p>Por favor, inicia sesión para continuar.</p>
                    </div>
                  )}
                </div>
              );
            }
            
            export default App;
💡Clave: Se usa parentesis () para agrupar varias líneas dentro del ternario. ❗❗

Ejemplo 2: Usando una estructura if antes del return

En este caso, se define la variable content antes del return.


            function App() {
              const isLoggedIn = true;
              let content;
            
              if (isLoggedIn) {
                content = (
                  <div>
                    <h1>Bienvenido, Usuario</h1>
                    <p>Nos alegra verte de nuevo.</p>
                  </div>
                );
              } else {
                content = (
                  <div>
                    <h1>Acceso Denegado</h1>
                    <p>Por favor, inicia sesión para continuar.</p>
                  </div>
                );
              }
            
              return <div>{content}</div>;
            }
            
            export default App;
💡 Clave: Definimos el contenido antes del return y lo usamos en JSX. ❗❗

Ejemplo 3: Usando un componente separado

Cuando la estructura es grande, es mejor crear un componente separado.


            function UserProfile() {
              return (
                <div>
                  <h1>Bienvenido, Usuario</h1>
                  <p>Nos alegra verte de nuevo.</p>
                </div>
              );
            }
            
            function GuestMessage() {
              return (
                <div>
                  <h1>Acceso Denegado</h1>
                  <p>Por favor, inicia sesión para continuar.</p>
                </div>
              );
            }
            
            function App() {
              const isLoggedIn = false;
            
              return <div>{isLoggedIn ? <UserProfile /> : <GuestMessage />}</div>;
            }
            
            export default App;
