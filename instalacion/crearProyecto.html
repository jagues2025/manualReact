Crear un proyecto react

Para crear un proyecto react, primero debes tener instalado node.js en tu computadora. Si no lo tienes instalado, puedes descargarlo desde aquí.

Una vez que tengas instalado node.js, puedes crear un proyecto react ejecutando el siguiente comando en tu terminal:

npm create vite@latest

Después de ejecutar este comando, se te pedirá que ingreses el nombre de tu proyecto. Puedes ingresar el nombre que desees y presionar enter.

Puede dejar el nombre del package con el mismo nombre del paquete

Seleccionamos el framework en el que se desea trabajar en este caso es React

Seleccionamos el lenguaje en el que se desee trabajar en este caso selecciono JavaScript

Despues en la terminal escribimos "cd nombre_carpeta"

Despues ejecutamos el comando "npm install" para generar la carpeta "node_modules"

Despues ejecutamos el comando "npm run dev" paa ejecutar el servicio
