¿Qué es la interpolación en React?

La interpolación en React es el proceso de insertar valores dinámicos (como variables, expresiones o resultados de funciones) dentro de JSX. Esto se logra utilizando llaves {} dentro del JSX. Las llaves permiten incluir cualquier expresión de JavaScript válida, como variables, operaciones matemáticas, resultados de funciones, etc.

Reglas Básicas de Interpolación

1. Las llaves {} se usan para encapsular expresiones de JavaScript.

2. Solo se pueden usar dentro del JSX (no fuera de él).

3. No funcionan para declaraciones completas como if o for (usa ternarios o métodos como .map() para eso).

Ejemplo 1


                function App() {
                    const name = "Miguel";
                    const age = 25;
                  
                    return (
                      <div>
                        <h1>Hola, {name}!</h1>
                        <p>Tienes {age} años.</p>
                      </div>
                    );
                  }
                  
                  export default App;

Ejemplo 2


                function App() {
                    const isLoggedIn = false;
                    const userName = "Miguel";
                  
                    return (
                      <div>
                        <h1>{isLoggedIn ? `Hola, ${userName}` : "Por favor, inicia sesión"}</h1>
                        {isLoggedIn && <p>¡Nos alegra verte de nuevo!</p>}
                      </div>
                    );
                  }
                  
                  export default App;
